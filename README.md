A powerful, web-responsive administrative dashboard designed to manage the **FoundIt Global Classifieds Platform**. This tool provides platform moderators and administrators with complete control over users, listings, and core system configurations.

This repository contains the standalone Flutter Web codebase, connected directly to the FoundIt Firebase backend.

## ✨ Key Features

- **Real-Time Analytics:** Track total users, pending approvals, active listings, and category counts on a clean overview dashboard.
- **Category Management:** A dynamic interface to add, edit, or remove the classifieds categories (e.g., Motors, Property, Electronics) that populate the main app.
- **Listing Moderation:** Review recent activity and manage user product listings directly from the data tables.
- **Secure Access:** Locked behind Firebase Authentication, ensuring only authorized administrators can view or alter platform data.
- **Web-Optimized UI:** Built specifically for desktop browsers with a persistent sidebar navigation and expansive data tables.

## 🛠️ Built With

* **Framework:** [Flutter](https://flutter.dev/) (Web)
* **Backend:** [Firebase](https://firebase.google.com/) (Auth & Cloud Firestore)
